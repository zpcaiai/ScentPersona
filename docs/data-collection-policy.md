# ScentPersona 商品数据采集合规守则

允许的数据来源：

- 官方开放平台 API
- 联盟 API
- 商家授权接口
- 用户上传 CSV
- 手动录入
- 明确允许访问的公开页面数据

禁止的数据来源和行为：

- 绕过登录、验证码、反爬或风控
- 调用未授权私有接口
- 逆向 App 私有接口
- 使用盗取或共享 cookie
- 高频请求造成平台负担
- 抓取用户个人数据
- 存储平台账号、密码、cookie 等敏感凭据

实现要求：

- 所有平台密钥只从环境变量读取。
- 商品报价必须保留 `platform`、`sourceUrl`、`fetchedAt` 和 `rawDataJson`。
- 价格、优惠、库存、评分和销量均需展示来源和更新时间。
- 页面必须提示“价格、优惠和库存可能变化，请以平台页面为准。”
- 未通过合规检查的数据源不能执行任务。
